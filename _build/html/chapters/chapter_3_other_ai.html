3. Finding the Best Model for Your Task#

How do I know which model(s) is best for what task?

3.1 Chatbot Arena#

LMArena is a battle field for different large language generative models. Everytime you prompt a question, 2 anonymous AI model will be asked to answer your question, and you will vote which model gave better response. After you give your vote, the system will reveal the models‚Äô names. After millions of pairwise comparisons are made, an Elo-rating algorithm is used to rank the AI models based on people‚Äôs preferences.

Text-to-image arena uses the same logic, but it is a battle field for different text-to-image generative models.

  • Free to try all models

  • You can vote and contribute to rankings!

  • Great for comparing model performance

8. Prompt Engineering Tips#

8.1 üìù The Power of Note-Taking & System Prompts#

Who loves taking notes, organizing class notes? üôã‚Äç‚ôÄÔ∏è

üéØ The Power of System Prompts

AI can get lost in the woods with too much information it reads all over the internet. System prompts allow the AI agent to focus on solving specific tasks with best practices.

üß≠ System prompts allow you to be the guide for AI - you become the navigator!

üí° Pro Tip:

  • When you learn a new concept or workflow, write notes for yourself

  • But also design system prompts for AI

  • This highlights the importance of communication skills with humans AND with AI ‚Üí prompt engineering!

‚ö†Ô∏è Always fact-check against textbooks or credible resources - AI can hallucinate!

8.2 Core Techniques#

  • Ask AI to think step by step - Break down complex problems

  • Give AI time to think - ‚ÄúTake your time and think carefully before answering‚Äù

  • Use structured formats - Request JSON, numbered lists, or bullet points for answers

  • Give AI a persona - ‚ÄúYou are an expert data scientist with 20 years of experience‚Äù

8.3 Best Practices#

  • Provide clear examples - Show exactly what format or style you want

  • Ask for explanations - Don‚Äôt just get answers, ask about the ‚Äúwhy‚Äù behind them

  • Use fun analogies - ‚ÄúExplain MCP using LEGO building blocks‚Äù

  • Ask for alternatives - Get multiple options: ‚ÄúShow me 3 different ways to solve this‚Äù

8.4 Communication Tips#

  • Set clear constraints - ‚ÄúYou can only answer in 200 words‚Äù

  • Be polite - Good manners really do help! Say ‚Äúplease‚Äù and ‚Äúthank you‚Äù

  • Be super specific - More details = better answers

  • Ask questions in JSON format - AI loves structured requests

8.5 Context Management#

  • Manage context window - Start new chats for big tasks

  • Avoid overwhelming AI - AI can get overwhelmed with 20+ messages
